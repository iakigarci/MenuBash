# Menú en Bash

Proyecto individual de la asignatura de "Sistemas Operativos", consiste en un menú para consola que dispone de diferentes funcionalidades.

## Uso
Dispone de un menú principal que permite seleccionar diferenentes funcionalidades:

1. Desinstalar
2. Instalar Apache
3. Activar Apache
4. Instalar PHP y comprobar su funcionamiento
5. Instalar Python3
6. Instalar numpy, nltk, argparse, dos2unis y python3-pip.
7. Testear Python en VM.
8. Comprobar complejudad
9. Visualizar complejidad
10. Mostar los errores y logs ocurridos durante la ejecución
11. Configuración de SSH

Para utilizarlo, es necesario utilizar el siguiente comando en la terminal

`bash -x menu.sh`

## Herramientas
Aplicación desarrollada en el SO Linux (Debian 9 Jessie), con el uso de **Python3** y **bash**.
